Harpa clustering

Acerca de HCluster

Microservicio creado para generar una clasificación de n cantidad de grupos en base a la técnica de clustering que identifica de forma automática agrupaciones de elementos de acuerdo a su estructura interna (propiedades o características), a través del uso del algoritmo K-means*.

*Entendido como K-means al algoritmo que tiene por objetivo descubrir agrupamientos en un conjunto de datos.

Aplicaciones

  • Algoritmo de clusterización
  • Minería de datos
  • Aprendizaje no supervisado

Especificación técnica

  • Servicios: WEB API
  • URL: harpa.hde.mx/harpa/hcluster
  • Métodos: HTTP REST POST

Nota: Este método puede ser combinado con otros métodos de análisis de datos para un mayor enfoque y estudio de la información requerida.

Instrucciones Cuerpo de la petición del microservicio


Se cuenta con un conjunto de datos señalados en dos vectores denominados respectivamente Variables Y y X, en el que se generará la clasificación en cantidad de grupos determinada, los cuales seran utilizados como características de entrada:

{
 "vars":
    {
     "Variable X":[1,2,3,4,5],
     "Variable Y":[2,4,6,8,10]
    },
 "grupos":2
 }

Una vez proporcionados los datos se da inicio a la clusterización usando el algoritmo de K-means quien se encargara de agrupar las observaciones por sus similitudes, de modo que el sistema muestre como resultado los grupos clasificados como características de salida:


{
 "exec":
  {
   "funcion": "HCluster",
   "horafin": "2019-04-09 13:38:26.197244",
   "horainicio": "2019-04-09 13:38:26.187308",
   "tiempoexec": "0:00:00.009936"
  },
  "result":
  {
   "attach": {},
   "vars":
   {
    "cluster": [0,0,1,1,1]
   }
  }
 }
