Glosario

Término Significado
API
(Application Programming Interface)

Permite hacer uso de funciones ya existentes en otro software.
POSTMAN

Extensión gratuita para el navegador Google Chrome que permite probar servicios web, requiriendo la URL, el método HTTP (POST, GET, etc.) y los parámetros de
la petición.
REST

Entendido como cualquier interfaz entre sistemas que use HTTP para obtener datos o generar operaciones sobre esos datos en formatos, como XML y JSON.
Minería de datos

Campo de la estadística referido al proceso que intenta descubrir patrones en grandes volúmenes de conjuntos de datos, mediante el uso de métodos de la inteligencia artificial, aprendizaje automático, estadística y sistemas de bases de datos.
Aprendizaje automático

Rama de inteligencia artificial y subcampo de ciencia de la computación en la que se busca crear programas capaces de generalizar comportamientos a partir de una información suministrada en forma de ejemplos.
Inteligencia Artificial

Programa de computación diseñado para realizar determinadas operaciones que se consideran propias de la inteligencia humana, como el autoaprendizaje.
Regresión lineal

Modelo matemático usado para aproximar la relación de dependencia entre una variable dependiente Y, las variables independientes Xi y un término aleatorio ε, con la finalidad de encontrar el valor esperado de una variable aleatoria (Y) cuando la variable X toma un valor específico.
Diagrama de dispersión

Diagrama matemático en el que se muestran valores de dos conjuntos asociados de datos que buscan analizar la existencia de algún tipo de relación entre dos variables para determinar el efecto en una variable ante los cambios producidos en la otra.
