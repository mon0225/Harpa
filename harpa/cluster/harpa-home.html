Acerca de Harpa

Sistema integrado por un conjunto de microservicios (API's) con funcionalidad analítica, dividida en tres servicios:

N° Microservicio Link
1 HCluster
(Clusterización)
2 HRegLinSimple
(Regresión lineal simple)
3 HRegLinMúltiple
(Regresión lineal múltiple)
4 HTransfV
(Transformación de variable)

Aréas de conocimiento

  • Minería de datos.
  • Machine Learning.
  • Deep Learning.

Características

  • Tipo de servicio: WEB API
  • Métodos:HTTP POST
